# Spouse Example

This directory contains various implementations of the spouse example used in [our walk-through](http://deepdive.stanford.edu/doc/basics/walkthrough/walkthrough.html).

* PostgreSQL (`postgres/`)
    * `tsv_extractor/`
    * `json_extractor/`
    * `plpy_extractor/`

* Greenplum (`greenplum/`)
    * `tsv_extractor/`

* Postgres-XL (`pgxl/`)
    * `tsv_extractor/`

* MySQL (`mysql/`)
    * `tsv_extractor/`

* MySQL Cluster (`mysql/`)
    * `tsv_extractor/`
